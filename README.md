## Get started

1. [Install DocPad](https://github.com/bevry/docpad)

1. Clone the project and run the server

	``` bash
	git clone git://github.com/iredmedia/iredmedia.git
	cd iredmedia
	npm install
	docpad run
	```

1. [Open http://0.0.0.0:9778/](http://0.0.0.0:9778/)

1. Start hacking away by modifying the `src` directory
